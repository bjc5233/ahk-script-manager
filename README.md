# ahk-script-manager
> AHK脚本管理工具

### 说明
1. 启动时, 执行配置中的ahk脚本
2. 启动的ahk脚本**不会展示托盘图标**
3. 右键菜单有[刷新][新增][启动][退出][删除][定位文件]
